## рЯТ° Key Services Overview

### RouteGeometryService
**What**: Distance and geometry calculations
**When**: Need to calculate distances or detect deviations
**Key Methods**: calculateHaversineDistance(), projectToRoute(), isOffRoute()

### RouteProgressService
**What**: Progress tracking along route
**When**: Need to get progress information
**Key Methods**: calculateProgress(), getRemainingPathNodes()

### RouteStateService
**What**: Session lifecycle management
**When**: Control navigation (start, pause, resume, complete)
**Key Methods**: startRoute(), pauseRoute(), resumeRoute(), completeRoute()

### RouteAlertService
**What**: Destination approach alerts
**When**: Create/check alerts for navigation
**Key Methods**: createDestinationAlert(), checkAndTriggerAlerts()

### RouteCalculationService
**What**: Route planning and rerouting
**When**: Plan a route or reroute user
**Key Methods**: planRoute(), rerouteFromPosition(), storeRouteNodes()

### RouteMovementService
**What**: Movement detection and speed calculation
**When**: Detect if user is moving
**Key Methods**: calculateMovementMetrics(), isUserMoving(), resolveSpeed()

### RouteService
**What**: Main coordinator and entry point
**When**: Use public API for route management
**Key Methods**: planRoute(), updateLocation(), startRoute(), pauseRoute(), etc.
